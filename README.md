# 🧮 Calculadora Nutricional em Python

Aplicação desenvolvida em Python para cálculo de indicadores nutricionais essenciais, permitindo ao usuário calcular IMC, TMB e GET através de interação via terminal.

Projeto criado com foco em prática de lógica de programação, organização de funções e aplicação de fórmulas utilizadas na área de nutrição e fitness.

---

## 🚀 Funcionalidades

- ✅ Cálculo de IMC (Índice de Massa Corporal)
- ✅ Classificação automática do IMC
- ✅ Cálculo de TMB utilizando a fórmula Mifflin-St Jeor
- ✅ Cálculo de GET baseado no fator de atividade diária
- ✅ Menu interativo via terminal

---

## 🧠 Fórmulas Utilizadas

### IMC
IMC = Peso / Altura²

### TMB (Mifflin-St Jeor)

Homens:

TMB = (10 × peso) + (6.25 × altura) − (5 × idade) + 5

Mulheres:

TMB = (10 × peso) + (6.25 × altura) − (5 × idade) − 161

### GET

GET = TMB × Fator de Atividade
