# 🧮 Calculadora Nutricional em Python

Projeto desenvolvido em Python com o objetivo de calcular indicadores nutricionais importantes utilizando fórmulas reconhecidas.

## 📌 Funcionalidades

- Cálculo de IMC (Índice de Massa Corporal)
- Cálculo de TMB (Taxa Metabólica Basal)
- Cálculo de GET (Gasto Energético Total)

## 🧠 Fórmulas Utilizadas

- IMC = Peso / Altura²
- TMB = Fórmula Mifflin-St Jeor
- GET = TMB × Fator de Atividade

## 💻 Tecnologias

- Python 3
